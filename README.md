# homebrew-tap

Homebrew tap for [container-primer](https://github.com/andreichenchik/container-primer).

```bash
brew install andreichenchik/tap/container-primer
```

Apple silicon Macs on macOS 26+. The formula installs a prebuilt, signed binary; release
artifacts live in the container-primer repo. The formula is bumped automatically by
`make release` there.
